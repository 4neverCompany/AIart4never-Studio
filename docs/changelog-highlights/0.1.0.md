# 0.1.0 — Foundation

First public build of **AIart4never Studio** — 4neverCompany's autonomous AI
content agent for the AIart4never channel (Master4never / Kael).

## 🎬 Highlights

### MiniMax-only engine
The desktop studio runs entirely on a single **MiniMax** subscription (the flat
Token-Plan, not pay-as-you-go). Text, planning, and tool-calls go through
MiniMax-M3 via the OpenAI-compatible Chat Completions API; image/video via
MiniMax. No other LLM, no per-call billing surprises.

### Monthly token-quota tracking
Tracks your MiniMax token consumption against your tier's monthly allowance,
warns at 90%, and pauses the autonomous loop before the subscription is
exhausted. Tier picker + custom cap in **Settings → Token Quota**.

### Signed Windows installer + auto-update
NSIS installer with minisign-signed auto-update, built and released from CI.

## 📋 What this release is (and isn't)
v0.1.0 is the **foundation** milestone (M0): the proven content-pipeline engine,
rebranded and hardened to MiniMax-only, with the build → sign → update pipeline
validated end-to-end. The on-canon Master4never generation engine, the in-app
Connectors & Skills (MCP) manager, and the growth/analytics brain land across
M1–M3.
